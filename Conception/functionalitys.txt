as a mimt Proxy : the program will implement some server side functionalitys , and some client side functionalitys
Starting with the Server Side (for the browser) the program must implement : 
* tcp handshake with the browser 
* ssl handshake in case of https 
* parse and verify the http request structure
* decrypt an http request 
* encrypt an http response
* send http response 
For the Client Side (for the website) the program must implement : 
* tcp handshake with the website 
* ssl handshake with the website in case of https 
* encrypt an http request 
* send and http request
* receive and http/s response 
* decrypt the https response
* parse the and verify the http/s response
